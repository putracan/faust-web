Blueprint eksekusi

Konten prioritas seluler + data MySQL sinkron otomatis

Kami memulai dari layout 360px, memastikan UI Tailwind + shadcn tetap ringan, lalu menghubungkan setiap aksi ke database MySQL sehingga dashboard manajemen, CCTV, hingga modul HR selalu up-to-date tanpa proses manual.

  • Grid responsif dengan clamp spacing & CSS variable shadcn.
  • Critical CSS + lazy loading gambar agar LCP < 2.5 detik.
  • Webhook ke MySQL/Postgre untuk auto update ketika teknisi mengirim laporan via ponsel.
  • Service worker dengan strategi stale-while-revalidate dan backup offline checklist inspeksi.
